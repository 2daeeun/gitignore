# 자주 사용하는 gitignore파일
매번 로컬에서 gitignore파일 찾는게 귀찮아서 그냥 올려둠.

### 빠른 다운로드
```swift
wget https://raw.githubusercontent.com/2daeeun/gitignore/main/.gitignore
```
### 내용
```swift
# Ignore all
*

# Unignore all with extensions
!*.*

# Unignore all dirs
!*/

# Ignore ccls-cache directory
.ccls-cache

# Ignore ctags
tags
```
